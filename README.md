# Nexora Blog

Nexora Blog is a modern social blogging platform that allows users to create and share posts, interact through comments, follow other users, and build personalized profiles.

The platform is designed to provide an engaging environment where users can publish content, connect with others, and participate in community discussions.

---

## Features

* User Registration & Login System
* Create, Edit & Delete Posts
* Comment System
* Follow & Unfollow Users
* Personalized User Profiles
* Profile Picture Upload
* Dynamic Home Feed
* Responsive Design
* Secure Authentication
* Database-Driven Content Management

---

## Technologies Used

* PHP
* MySQL
* HTML5
* CSS3
* JavaScript
